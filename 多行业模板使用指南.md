## 🎯 快速开始

### 一行命令初始化

```bash
npm run setup:industry manufacturing
```

### 支持的行业

| 命令 | 行业 | 适用场景 |
|------|------|---------|
| `manufacturing` | 🏭 制造业 | LED、电子、机械 |
| `service` | 💼 服务业 | 咨询、法律、会计 |
| `retail` | 🛍️ 零售业 | 电商、品牌 |
| `technology` | 💻 科技行业 | 软件、SaaS、AI |

## ⚙️ 核心配置文件

### 1. `lib/config/industry.ts` - 行业配置

```typescript
export const SITE_CONFIG: SiteConfig = {
  siteName: 'Your Company',      // 公司名称
  siteDescription: 'Professional solutions',
  industry: 'manufacturing',     // 行业类型
  primaryColor: '#3B82F6',       // 品牌色
  locales: ['zh', 'en'],         // 支持的语言
  
  enabledFeatures: {
    products: true,              // 产品中心
    news: true,                  // 资讯中心
    geoSeo: true,                // SEO 优化
    contactForm: true,           // 联系表单
  },
};
```

### 2. `.env.local` - 环境变量

```env
NEXT_PUBLIC_SANITY_PROJECT_ID=你的 project_id
NEXT_PUBLIC_SANITY_DATASET=production
SANITY_API_TOKEN=你的 api_token
NEXT_PUBLIC_SITE_URL=https://yourdomain.com
```

### 3. `template.config.json` - 模板元数据

定义所有可用的功能模块和行业预设。

## 🔧 自定义步骤

### Step 1: 修改品牌信息

**文件**: `lib/config/industry.ts`

```typescript
siteName: '你的公司名称',
siteDescription: '公司描述或标语',
primaryColor: '#你的品牌色',
```

### Step 2: 调整颜色主题

**文件**: `tailwind.config.js`

```javascript
theme: {
  extend: {
    colors: {
      primary: {
        500: '#3B82F6',  // 主色调
        600: '#2563EB',  // 深色
        700: '#1D4ED8',  // 更深色
      },
    },
  },
}
```

### Step 3: 配置 Sanity CMS

1. 访问 https://www.sanity.io/manage
2. 创建新项目
3. 获取 Project ID 和 API Token
4. 填入 `.env.local`

### Step 4: 添加内容

访问 http://localhost:3333 添加：
- 产品分类
- 产品/服务
- 关于页面
- 联系方式

## 🗑️ 移除非必须内容

### 如果不需要产品中心

编辑 `lib/config/industry.ts`：

```typescript
enabledFeatures: {
  products: false,  // 禁用产品中心
  // ... 其他保持不变
}
```

运行清理脚本（可选）：

```bash
npm run setup:industry service
```

会自动删除产品相关的 Schema 文件。

### 如果不需要资讯自动化

删除或忽略以下目录：
- `scripts/news-auto/` - 自动采集脚本
- `.env.local` 中的 `DASHSCOPE_API_KEY`

## 🌍 多语言支持

### 默认配置

- 中文 (zh)
- 英文 (en)

### 添加新语言

1. **编辑配置** `lib/config/industry.ts`：

```typescript
locales: ['zh', 'en', 'es', 'fr']  // 添加西班牙语、法语
```

2. **创建翻译文件** `messages/es.json`：

```json
{
  "navigation": {
    "home": "Inicio",
    "products": "Productos",
    "about": "Nosotros",
    "contact": "Contacto"
  }
}
```

3. **更新 Sanity Schema**（如果需要）：

在 `sanity/schemas/product.ts` 等文件中添加对应语言字段。

## 📦 部署选项

### Vercel（推荐）

**优点**：
- ✅ 零配置自动部署
- ✅ 全球 CDN 加速
- ✅ 自动 HTTPS
- ✅ 免费套餐充足

**步骤**：
1. 推送代码到 GitHub
2. 在 Vercel 导入项目
3. 配置环境变量
4. 自动部署

### Netlify

**步骤类似 Vercel**

### 自建服务器

```bash
npm run build
npm run start
```

需要配置 Node.js 环境和 PM2 进程管理。

## 🔍 SEO 优化

### 自动功能

- ✅ Meta 标签自动生成

- ✅ Canonical URLs
- ✅ Hreflang 标签
- ✅ Sitemap.xml

```typescript

  return {
    title: '自定义页面标题',
    description: '页面描述',
    keywords: ['关键词 1', '关键词 2'],
  };
}
```

## 🆘 故障排查

### 问题：Sanity Studio 无法加载

**解决**：
```bash
cd sanity
npm install
npm run dev
```

### 问题：环境变量不生效

**检查**：
1. 文件名是否为 `.env.local`（不是 `.env`）
2. 是否重启了开发服务器
3. 变量名是否正确（区分大小写）

### 问题：构建失败

**尝试**：
```bash
rm -rf .next
npm run build
```
