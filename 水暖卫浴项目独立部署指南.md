## 🚀 独立部署步骤

### Step 1: 进入项目目录

```bash
cd L:\gopro-plumbing-website
```

### Step 2: 安装依赖

```bash
npm install
```

### Step 3: 配置环境变量

```bash
# 复制模板
copy .env.plumbing.template .env.local

# 编辑 .env.local，填入真实的 API Key
notepad .env.local
```

需要配置的变量：

```env
# Sanity CMS
NEXT_PUBLIC_SANITY_PROJECT_ID=your_project_id
SANITY_API_TOKEN=your_token

# 阿里云百炼（AI 功能必需）
DASHSCOPE_API_KEY=sk-your_api_key

# 站点配置
NEXT_PUBLIC_SITE_URL=https://plumbcore.com

# 邮件服务
CONTACT_EMAIL=sales@plumbcore.com
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your_email@gmail.com
SMTP_PASS=your_app_password
```

### Step 4: 运行初始化脚本（可选）

```bash
node scripts/setup-plumbing.js
```

这会自动：
- 创建 `.env.local` 文件
- 更新行业配置文件
- 提供产品分类建议

### Step 5: 启动开发环境

```bash
# 启动网站（http://localhost:3000）
npm run dev

# 启动 Sanity Studio（http://localhost:3333）
npm run sanity
```

### Step 6: 配置 Sanity CMS

1. 访问 https://www.sanity.io/
2. 创建新项目或复用现有项目
3. 获取 Project ID 和 API Token
4. 在 Sanity Studio 中创建产品分类：
   - Faucets & Taps
   - Showers & Accessories
   - Toilets & Bidets
   - Basins & Sinks
   - Bathroom Furniture
   - Sanitary Ware
   - Plumbing Fixtures
   - Kitchen Solutions

### Step 7: 添加产品数据

在 Sanity Studio 中添加 5-10 个示例产品：
- 产品名称（中英文）
- 产品图片（至少 1 张）
- 描述和规格
- SEO 信息

### Step 8: 部署到 Vercel

```bash
# 安装 Vercel CLI（如果未安装）
npm i -g vercel

# 登录 Vercel
vercel login

# 部署
vercel --prod
```

或使用 GitHub 自动部署：
1. 创建新的 GitHub 仓库
2. 推送代码：`git init && git add . && git commit -m "Initial commit" && git push`
3. 在 Vercel 导入 GitHub 仓库
4. 配置环境变量
5. 自动部署

## 🔍 常见问题

### Q1: 如何切换到其他行业？

编辑 `lib/config/industry.ts`，选择不同的预设：
```typescript
export const SELECTED_INDUSTRY: IndustryPreset = PLUMBING_PRESET;
```

### Q2: 本地是否需要一直运行？

**不需要！** Vercel Cron 在云端执行，即使您电脑关机也能正常运行。

### Q3: AI 生图失败怎么办？

检查：
1. ✅ DASHSCOPE_API_KEY 是否正确配置
2. ✅ API 余额是否充足
3. ✅ Prompt 是否符合规范

### Q4: 如何关闭资讯自动发布？

编辑 `lib/config/industry.ts`：
```typescript
enabledFeatures: {
  newsAutoPublish: false,
}
```
