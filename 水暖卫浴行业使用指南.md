# 水暖卫浴行业项目使用指南
# Plumbing & Sanitary Ware Industry Guide

## 📋 快速开始 / Quick Start

### 方式一：自动初始化（推荐）

```bash
# 运行一键初始化脚本
node scripts/setup-plumbing.js

# 启动开发服务器
npm run dev
```

### 方式二：手动配置

1. **复制配置文件**
   ```bash
   cp .env.plumbing.template .env.local
   ```

2. **编辑环境变量** `.env.local`
   ```env
   NEXT_PUBLIC_SANITY_PROJECT_ID=your_project_id
   SANITY_API_TOKEN=your_token
   DASHSCOPE_API_KEY=sk-your_api_key
   NEXT_PUBLIC_SITE_URL=https://plumbcore.com
   ```

3. **更新行业配置** `lib/config/industry.ts`
   ```typescript
   import { PLUMBING_PRESET, PLUMBING_CONFIG } from './industry.plumbing';
   
   export const SELECTED_INDUSTRY: IndustryPreset = PLUMBING_PRESET;
   export const SITE_CONFIG: SiteConfig = PLUMBING_CONFIG;
   ```

## 🔧 常见问题 / FAQ

### Q1: 如何添加新产品？

1. 登录 Sanity Studio (`yoursite.com/studio`)
2. 点击 **Products** → **Create new**
3. 填写产品名称（中英文）
4. 选择分类
5. 上传产品图片（至少 1 张）
6. 填写描述和规格
7. 保存并发布

### Q2: 如何批量导入产品？

使用脚本批量导入：

```bash
# 准备 CSV/Excel 数据
# 运行导入脚本
npx tsx scripts/import-products.ts
```

### Q3: AI 生图失败怎么办？

检查以下几点：
1. ✅ DASHSCOPE_API_KEY 是否正确配置
2. ✅ API 余额是否充足
3. ✅ 网络连接是否正常
4. ✅ Prompt 是否符合规范（英文、具体描述）

### Q4: 如何关闭资讯自动发布？

编辑 `lib/config/industry.ts`：

```typescript
enabledFeatures: {
  newsAutoPublish: false,  // 关闭自动发布
}
```
